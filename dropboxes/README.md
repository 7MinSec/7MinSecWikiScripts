# dropboxes/

Scripts to automate deploying **Proxmox VMs** that serve as pentest dropboxes —
the remote-access boxes you drop on a client network for an engagement.

| Script | Purpose | Wiki |
|--------|---------|------|
| [`install-exegol.sh`](install-exegol.sh) | One-shot [Exegol](https://exegol.com) installer for Kali. Idempotent; run as a normal user with sudo rights. | [7minsec.wiki/software/exegol](https://7minsec.wiki/software/exegol/) |
